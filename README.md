# liba

Library of ASH functions for use in scripts in Kolmafia. Current focus is on IotMs that are new and/or don't have their own native Kolmafia functions.

## Installation

Can be installed from the CLI in Kolmafia via:

```
git checkout https://github.com/C2Talon/liba.git master
```

## Things supported

This library has functions supporting the following things:

* 2013
	* [warbear LP-ROM burner](https://wiki.kingdomofloathing.com/Warbear_LP-ROM_burner)
* 2017
	* [genie bottle](https://wiki.kingdomofloathing.com/Genie_bottle)
* 2022
	* [unbreakable umbrella](https://wiki.kingdomofloathing.com/Unbreakable_umbrella)
* 2024
	* [Clan Photo Booth](https://wiki.kingdomofloathing.com/Clan_Photo_Booth)
* 2025
	* [Allied Radio Backpack](https://wiki.kingdomofloathing.com/Allied_Radio_Backpack)
	* [April Shower Thoughts shield](https://wiki.kingdomofloathing.com/April_Shower_Thoughts_shield)
	* [blood cubic zirconia](https://wiki.kingdomofloathing.com/Blood_cubic_zirconia)
	* [Cooler Yeti](https://wiki.kingdomofloathing.com/Cooler_Yeti)
	* [Leprecondo](https://wiki.kingdomofloathing.com/Leprecondo)
	* [Peridot of Peril](https://wiki.kingdomofloathing.com/Peridot_of_Peril)
	* [prismatic beret](https://wiki.kingdomofloathing.com/Prismatic_beret)
